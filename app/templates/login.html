What this system does

This security monitoring system checks login attempts and captures the client IP address for every request.

If a login fails, it records the email, IP address, and timestamp in login.log for review.

Repeated failures from the same IP are tracked to help detect suspicious activity such as brute-force attacks.

Alert emails are sent by a separate local notifier app, so Gmail credentials stay off the deployed server.
